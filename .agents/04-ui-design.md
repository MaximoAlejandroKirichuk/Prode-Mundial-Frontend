# Skill: UI Design

## Use When
- Se solicita mejora visual, layout o nueva interfaz.
- Se busca pulido visual o diseno distintivo.

## Goal
Crear UI clara y coherente sin desviarse del lenguaje visual actual.

## Load Next Only If Needed
- `.agents/references/repo-map.md`
- archivos fuente afectados

## Design Rules
- Respetar el estilo actual salvo que se pida redisenio.
- Usar Tailwind si el area ya lo utiliza; si hay CSS local, mantener el patron.
- Evitar layouts genericos o excesivamente templados.
- Diseñar para mobile y desktop desde el inicio.
- Usar animacion solo si aporta feedback claro.


## Output
- direccion visual
- areas UI afectadas
- restricciones respetadas
- consideraciones responsive
- consideraciones de accesibilidad
